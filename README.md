## How to use

1. Start up docker.
    ```
    $ cd ./front
    $ docker-compose up
    ```
    Open another terminal.
    ```
    $ cd ./server
    $ docker-compose up
    ```
2. Open browser. Enter in `localhost:1234`.
3. Set MongoDB config.
    ```
    Connection name: Any name
    Connection string: mongodb://root:example@mongo:27017
    ```
